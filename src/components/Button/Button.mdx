# Button

<Meta of={ButtonStories} />

Buttons communicate actions that users can take. Although called “Buttons”, the actual HTML element that is rendered will depend on the action. onClick events will render as Buttons, but if provided an href, the component will render an anchor tag instead.

Note: Text & Inline are only different by padding, and have no semantic distinction
